# 学んだこと
## Docker ボリュームと永続化
volumes:
- コンテナが消えてもデータを保持したい場合に使う
```
volumes:
  - ruby-bundle-cache:/bundle
  - postgres_data:/var/lib/postgresql/data
```

## ports
- 左側: ホスト、右側: コンテナ →基本的にはこれを意識しておけばおけ
- RailsサーバーをMacからアクセス可能にする
```
ports:
  - "3000:3000"
```
