# 学んだこと
## pnpm と npm の違い
- pnpm は依存関係の管理を高速化・効率化するパッケージマネージャ
- pnpm install --frozen-lockfile → lockfile に書かれた正確なバージョンでインストール
- 開発中は必須でないが、本番デプロイ前に行うと安全
