# 学んだこと
## Git / GitHub 大きなファイルはpushが失敗する
- GitHub は 100MB を超えるファイルを直接 push できない
- .pnpm-store のような巨大キャッシュファイルは LFS（Git Large File Storage）で管理するか、そもそも Git 管理から除外する

## Docker コマンド
- docker compose exec 名前 <command> → 起動中コンテナにコマンド送信
- docker compose run 名前 <command> → 新しいコンテナを作ってコマンド実行
