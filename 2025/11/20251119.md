# 学んだこと
## useRouter
- router.push(path) ページ遷移
- router.replace(path)　履歴残さず遷移
- router.back()　戻る
- router.refresh() 現在のペーシを再フェッチ

現在のパスを取得するコード
```
import { usePathname } from 'next/navigation'

const pathname = usePathname()
```
