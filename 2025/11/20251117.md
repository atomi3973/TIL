# 学んだこと
## useEffectの実行タイミング
- useEffectはレンダリング（DOM 反映・画面描画）が完了した後に実行される
- そのためUIの描画を邪魔しない（非同期的に実行される）
- DOMを読み取る必要がある処理や、外部 API との通信に向いている

## useEffectの使用
- 基本的に使用しない方がいい。
- 外部との通信の時だけにする。
