# 学んだこと
## CDN
- 「コンテンツ配信ネットワーク」
- 世界中のサーバーにコピー（キャッシュ）を置くことで、ユーザーの近くから高速に配信する仕組み。

## Next.js Link
- ページ遷移を「SPA的に」行うためのコンポーネント。
- ページ全体をリロードせず、クライアント側で差分を更新。
- サーバーコンポーネントでも使用できる（クリック時はクライアントJSが反応）。

## RSC（React Server Components）
- React 18からの新概念。
- コンポーネントをサーバーで実行して、その結果（構造データ＝RSCペイロード）をクライアントに送る。
- HTMLだけでなく、RSCデータも同時に生成される（SSRでも同様）。

## FQDNとドメイン名
- FQDN（完全修飾ドメイン名）＝ホスト名＋ドメイン名＋ルートドメイン。
例：www.example.com.
- ドメイン名だけ（example.com）もDNS的にはFQDN扱い（. が省略されているだけ）。
- URLには「ドメイン名」だけでOK。
