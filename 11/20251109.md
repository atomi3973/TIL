# 学んだこと
## 「`'page.tsx' のインポートを更新しますか？」」の意味
- ファイル名や場所を変えたときに、VSCodeが他のファイルのimportパスを自動で直すか確認しているメッセージ。



## <SheetTrigger asChild> 
- asChild を使うと 子要素（例：<Button>）をそのままクリックトリガーにできる。
- <button> の入れ子を避けるための機能。
```
<SheetTrigger asChild>
  <Button>開く</Button>
</SheetTrigger>
```

## Supabase公式がletを使っている理由
- 公式ドキュメントはプレーンなJavaScript向けなので、ESLintルールを気にせずlet を使っている。
- TypeScript + ESLint のプロジェクトではconstに直すのがベスト。
