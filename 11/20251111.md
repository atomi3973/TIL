# 学んだこと
## JSON.stringify()
- JavaScriptのオブジェクトを文字列（JSON）に変換する関数
- fetchのbodyにオブジェクトを送るときによく使う

## API_KEYと! 
- process.env.GOOGLE_API_KEY → 環境変数から API キーを取得
- ! は TypeScript の「Non-Null Assertion」
- 「ここは絶対 null/undefined じゃないよ」とコンパイラに教える
```
const apiKey = process.env.GOOGLE_API_KEY!;
```
