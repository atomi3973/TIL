# 学んだこと
## {[id: string]: string }
- 動的にキーを作成することができる
- idに任意のキーを入れることができる
- {[name: string]: string }のように別にidという名前である必要はない

## includes('')
```
"React入門".includes('React') // true
"React入門".includes('入門')   // true
"React入門".includes('Vue')   // false
"React入門".includes('')      // true
```
- includes('')はtrueになる
- 最初空文字、つまり''のみtrueになるかと思ったが、文字列の時点でtrueになる
