# 学んだこと
## display_meta_tags()
- Rails の meta-tags gem が提供するビュー用ヘルパー
- まとめて <meta> タグを生成してくれる
- 引数はハッシュ

例コード. 
```
def default_meta_tags
  {
    site: "MySite",
    reverse: true,
    title: "トップページ",
    description: "MySiteの説明文です",
    canonical: request.original_url,
    og: {
      title: :title,
      type: 'website',
      url: request.original_url,
      image: current_site.og_image_url(:ogp),
      site_name: :site,
      description: :description,
      locale: 'ja_JP'
    },
    twitter: {
      card: 'summary_large_image',
      site: '@mysite'
    }
  }
end
```
