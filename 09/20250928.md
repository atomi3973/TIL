# 学んだこと
## 正規表現と文字判定
```
S[i] =~ /\d/
```
- S[i]：文字列の i 番目の文字
- =~：正規表現マッチ（マッチした位置のインデックスを返す、マッチしなければ nil）
- /\d/：数字（0-9）にマッチ

## 配列とループ
```
(1..number).each do |n|
  # 繰り返し処理
end
```
- (1..number)：1 から number までの範囲
- .each do |n| ... end：範囲内の数字を順に処理
- .. は右端含む、... は右端含まない
