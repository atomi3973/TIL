# 学んだこと
## 1.タイマー・繰り返し処理
setInterval. 
- 指定した関数を 一定間隔で繰り返し実行
- 停止する場合 clearInterval(intervalId)
- 例
```
let intervalId = setInterval(() => {
  console.log("1秒ごとに実行");
}, 1000);
```
